### ‼️ MASSIVE REFACTOR
1.   <ins>SparkleSeeker:</ins> HTML entry/boot file that starts the game.
2.   <ins>GameEngine:</ins> Main game loop, startup flow, and top-level game orchestration.
3.   <ins>Vars:</ins> Shared runtime data, mutable state, and simple setters only.
4.   <ins>Config:</ins> Player options, accessibility, sound/fx settings, and shared tunable values.
5.   <ins>WinRulesConditions:</ins> Win logic, progression rules, score thresholds, and progression-related copy.
6.   <ins>Input:</ins> Player movement input only, including keyboard, pointer, and touch movement.
7.   <ins>Player:</ins> Player behavior, player state helpers, and player rendering.
8.   <ins>ParticleEngine:</ins> Sparkles, friends, enemies, pickups, and burst behavior/rendering.
9.   <ins>UI:</ins> HUD, menus, overlays, menu layout, and menu click/touch handling.

##### ♿️ A11Y OPTIONS
-    Sound: music/sfx (off-min-low-med-max).
-    Colorblind: Deuteranope, Protanope, Tritanope, Monochrome.
-    Movement: freetouch, joystick, dpad.
